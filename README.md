# Rainfall-Prediction-LSTM
# Rainfall Prediction using LSTM (Time Series Forecasting)

## Overview
This project focuses on predicting rainfall using Long Short-Term Memory (LSTM) networks, a type of recurrent neural network suitable for time series forecasting.

## Objective
To capture temporal patterns and dependencies in rainfall data for accurate prediction.

## Methodology
- Data preprocessing and normalization
- LSTM-based deep learning model
- Sequential time-series forecasting

## Features
- Handles temporal dependencies
- Captures seasonal trends
- Suitable for environmental modeling

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib
